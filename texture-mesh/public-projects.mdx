---
title: "Get Public Projects"
api: "GET /texture-mesh/public-projects"
---

### Description
Returns an array of the public 3D Asset projects

### Query Parameters
<ParamField query="page" type="string">
  The page number, 30 assets per page
</ParamField>


<RequestExample>
```bash cURL
curl -X 'GET' \
  'https://api.mirageml.com/texture-mesh/public-projects' \
  -H 'accept: application/json'
```
</RequestExample>
