title: "Get Run"
api: "GET /texture-mesh/run"

<ParamField header="Authorization" type="string" required hidden />
<ParamField header="x-api-key" type="string" required hidden />

### Description

A run is an instance of a texture mesh creation.
Returns the run data and generated textures.

### Query Parameters

<ParamField query="run-id" type="string" required>
  The id of the run
</ParamField>

<RequestExample>

```bash cURL
curl -X 'GET' \
  'https://api.mirageml.com/texture-mesh/run?run-id={}' \
  -H 'accept: application/json'
```

</RequestExample>
