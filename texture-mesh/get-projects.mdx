---
title: "Get Projects"
api: "GET /texture-mesh/projects"
---

<ParamField header="Authorization" type="string" required hidden />
<ParamField header="x-api-key" type="string" required hidden />

### Description
Returns an array of the texture mesh projects

### Query Parameters
<ParamField query="cursor" type="string">
  The pagination cursor
</ParamField>

<ParamField query="sort" type="string">
  The sort direction. Enum: `ascending`, `descending`
</ParamField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'https://api.mirageml.com/texture-mesh/projects' \
  -H 'accept: application/json'
```
</RequestExample>
