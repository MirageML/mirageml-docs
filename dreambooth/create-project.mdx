<ParamField header="Authorization" type="string" required hidden />
<ParamField header="x-api-key" type="string" required hidden />

### Description

Creates a dreambooth project

### Form Data

<ParamField body="init-word" type="string" required>
  Describes the word
</ParamField>

<ParamField body="train-iterations" type="number" default="800">
  Number of iterations to train
</ParamField>

<ParamField body="square-func" type="string" default="expand">
  Enum: `expand`, `crop`, `square`
</ParamField>

<ParamField body="prior-preservation" type="boolean" default="false">
  Should be true if working with faces
</ParamField>

<ParamField body="safety-filter" type="boolean" default="false">
  Automatically filters NSFW images
</ParamField>

<ParamField body="links" type="string">
JSON object of links to images to download.
Example:
```
{"0": "https://sabe.io/images/saturn.png", "1": "https://sabe.io/images/mars.png", "2": "https://sabe.io/images/earth.png"}
```
</ParamField>

<ParamField body="files[]" type="files"></ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  https://api.mirageml.com/dreambooth/project \
  -H 'Authorization: <AUTHORIZATION>' \
  -H 'Cache-Control: no-cache' \
  -H 'content-type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
  -H 'x-api-key: <API_KEY>' \
  -F init-word=<INIT_WORD> \
  -F train-iterations=<TRAIN_ITERATIONS> \
  -F square-fun=<SQUARE_FUNC> \
  -F prior-preservation=<PRIOR_PRESERVATION> \
  -F safety-filter=<SAFETY_FILTER> \
  -F 'files[]=<FILES>' \
  -F 'links={"0": "https://sabe.io/images/saturn.png", "1": "https://sabe.io/images/saturn.png"}'
```
</RequestExample>
