---
title: "Get Run"
api: "GET /dreambooth/run"
---

<ParamField header="Authorization" type="string" required hidden />
<ParamField header="x-api-key" type="string" required hidden />

### Description
A run is an instance of a dreambooth image generation.
Returns the run data and generated images for the run.

### Query Parameters

<ParamField query="run-id" type="string" required>
  The id of the run
</ParamField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'https://api.mirageml.com/dreambooth/run?run-id=<RUN-ID>' \
  -H 'accept: application/json' \
  -H 'Authorization: <AUTHORIZATION>' \
  -H 'x-api-key: <API_KEY>'
```
</RequestExample>
