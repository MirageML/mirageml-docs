---
title: "Public Dataset"
api: "GET /texture-mesh/public-projects"
---

### Description
Returns the public 3D assets created on Mirage

### Query Parameters
<ParamField query="page" type="string">
  The paginated page to obtain data
</ParamField>

<ParamField query="filter" type="string">
  A text filter to search projects based on prompt
</ParamField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'https://api.mirageml.com/texture-mesh/project?project-id={}' \
  -H 'accept: application/json'
```
</RequestExample>
