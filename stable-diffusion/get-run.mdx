---
title: "Get Run"
api: "GET /stable-diffusion/run"
---

<ParamField header="Authorization" type="string" required hidden />
<ParamField header="x-api-key" type="string" required hidden />

### Description
A run is an instance of a stable diffusion image generation.
Returns the run data and generated images for the run.

### Query Parameters
<ParamField query="runId" type="string" required>
  The id of the run
</ParamField>

<RequestExample>
```bash cURL
curl -X 'GET' \
  'https://api.mirageml.com/stable-diffusion/run?runId=6374f405-a7e3-437d-8ad1-bb553d3c28e0' \
  -H 'accept: application/json'
```
</RequestExample>
